> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Enable sharing call invitations as `nafaq://join?ticket=...` URLs that open the app and auto-join.

**Architecture:** Add `tauri-plugin-deep-link` for URL scheme registration. Frontend helpers wrap/unwrap tickets as URLs. Deep link listener in `useCall.ts` handles incoming URLs on both cold and warm launch.

**Tech Stack:** Tauri v2 deep-link plugin, Vue composables, existing Iroh ticket format.

### Task 3: Create ticket URL helpers

**Files:**
- Create: `app/composables/useTicketUrl.ts`

- [ ] **Step 1: Create the composable**

Create `app/composables/useTicketUrl.ts`:

```ts
const SCHEME = "nafaq:";
const HOST_PATH = "//join";

export function wrapTicketUrl(ticket: string): string {
  return `nafaq://join?ticket=${encodeURIComponent(ticket)}`;
}

export function unwrapTicket(input: string): string {
  try {
    const url = new URL(input);
    if (url.protocol === SCHEME && url.pathname === HOST_PATH) {
      const t = url.searchParams.get("ticket");
      if (t && t.length > 0) return t;
    }
  } catch {
    // Not a valid URL — treat as raw ticket
  }
  return input;
}
```

### Task 4: Add deep link listener to useCall

**Files:**
- Modify: `app/composables/useCall.ts:105-186` (inside `initCallListeners`)

- [ ] **Step 1: Add deep link handler**

In `app/composables/useCall.ts`, inside `initCallListeners()`, add after the `fetchNodeInfo()` call (after line 129) and before the `listen("peer-connected", ...)` call:

```ts
    // Deep link handler — join call when app is opened via nafaq:// URL
    try {
      const { onOpenUrl, getCurrent } = await import("@tauri-apps/plugin-deep-link");

      const handleDeepLink = (urls: string[]) => {
        if (state.value !== "idle" && state.value !== "waiting") return;
        for (const url of urls) {
          const t = unwrapTicket(url);
          if (t !== url) {
            joinCall(t);
            return;
          }
        }
      };

      // Warm launch — app already running
      onOpenUrl(handleDeepLink);

      // Cold launch — app opened by the URL
      const pending = await getCurrent();
      if (pending) handleDeepLink(pending);
    } catch {
      // Deep link plugin not available (e.g., dev mode in browser)
    }
```

The `joinCall` reference: this function is defined in the outer `useCall()` scope. Since `initCallListeners` is called from `useCall()`, we need to restructure slightly. Move the `joinCall` function definition **above** the `initCallListeners` call, or extract the join logic. The simplest approach: make `initCallListeners` accept `joinCall` as a parameter.

Change line 24 from:
```ts
    initCallListeners();
```
to:
```ts
    initCallListeners(joinCall);
```

Change line 105 from:
```ts
async function initCallListeners() {
```
to:
```ts
async function initCallListeners(joinCall: (t: string) => Promise<void>) {
```

Also add the import at the top of the deep link handler block:
```ts
const { unwrapTicket } = await import("./useTicketUrl");
```

Note: Nuxt auto-imports from composables, but inside a dynamic async context we use explicit import for clarity.
